LIV8 OS is a high-fidelity, device-agnostic AI Operating System designed to centralize and orchestrate agency operations. From headless onboarding to multi-staff conversational hubs, LIV8 OS transforms the CRM experience into a unified neural command center - one main brain that connects to many tools.

## üöÄ Key Modules

### [Neural Staff Hub]
A tri-tier communication hub for **AI Staff**, **Human Staff**, and **CRM Contacts**. Includes a sleek iMessage-style inbox and high-fidelity Employee Cards for agent management and training.

### [Headless Builder Engine]
Automated "Construction" phase during onboarding:
- **Snapshot Deployment**: One-click deployment of SMS/Missed Call frameworks.
- **Funnel Provisioning**: Mapping brand DNA to high-conversion funnel templates.
- **Knowledgebase Extraction**: Deep scraping of business assets into the Neural Core.

### [Mobile Neural Integration]
**iOS Shortcuts + Voice-to-Command**. Record a voice note on your iPhone, and LIV8 OS processes the command via webhooks, performing the action in your CRM and returning spoken feedback.

### [Brand Hub & Social Planner]
Centralized **Asset Library** for brand identity persistence and a **Social Media Planner** for orchestrating AI-driven content generation across locations.

## üõ†Ô∏è Setup & Deployment

**Prerequisites:** Node.js, Vercel CLI (optional for production)

1.  **Clone & Install**:
    ```bash
    npm install
    cd dashboard-standalone && npm install
    ```
2.  **Environment Sync**:
    Set your `GEMINI_API_KEY` in `dashboard-standalone/.env` or `.env.local`.
3.  **Local Execution**:
    ```bash
    npm run dev
    ```
4.  **Production Deployment**:
    ```bash
    vercel --prod
    ```
